# 风暴钟塔

《风暴钟塔》是一款原创 HTML5 Canvas 纵向平台跳跃游戏。玩家扮演戴着黄铜风暴面具的信使“妮玛”，攀登被雷雨遗弃的古老钟塔，回收报时齿轮，并在塔顶重新唤醒风暴之钟。

## 启动

要求 Node.js 20+。

```bash
npm install
npm run dev
```

开发服务器默认为 `http://127.0.0.1:4173/`。

构建与测试：

```bash
npm test
npm run build
```

构建产物位于 `dist/`。

## 操作

- `A / D` 或 `← / →`：地面移动，空中有限方向修正
- 按住 `Space`：蓄力；松开后跳跃
- `Esc`：暂停或继续
- `R`：快速重新开始

蓄力越久，垂直冲量和定向水平冲量越大。空中修正使用独立预算，无法无限改变方向。每次坠落或被旋转钟摆击中都会回到最近检查点，坠落次数保留且总时间增加 8 秒。

## 内容

- 5 个连续区域：沉钟门厅、失速齿轮厅、裂风竖井、霜封档案室、雷冠钟室
- 普通、移动、碎裂、弹簧、单向、结冰与检查点平台
- 横风、旋转钟摆、结冰惯性三类机制
- 12 枚可收集报时齿轮与 2 个中途检查点
- 摄像机纵向跟随、三层以上视差、程序化雨幕与闪电
- 程序化 Web Audio 反馈，无外部音频依赖
- 用时、坠落次数、最高高度、收集数量与 `localStorage` 最佳记录
- 对异常坐标、越界和深度坠落自动恢复

## 代码结构

- `src/game.ts`：游戏状态机、固定时间步、输入编排和关卡事件
- `src/physics.ts`：蓄力、跳跃冲量、平台碰撞与旋转机关碰撞
- `src/render.ts`：Canvas 渲染、视差背景、角色动画、HUD 和结算界面
- `src/level.ts`：五区平台、收集物、检查点、旋转机关和风场定义
- `src/audio.ts`：本地合成音效与风声
- `src/storage.ts`：最佳成绩读取、比较和保存
- `src/checkpoint.ts`：检查点推进与失败惩罚的纯逻辑
- `tests/`：蓄力、碰撞、检查点与成绩保存测试

## 原创与依赖

角色、关卡布局、图形、动画和程序化声音均为本项目原创；未使用游戏引擎、第三方物理引擎或远程美术资源。运行时代码仅使用浏览器标准 API。
